**Scenario Definition**
Let $\mathbb{A}$ denote a finite action set.
A \emph{scenario} is defined as a tuple
\[
\mathcal{S} = (\mathbf{A}, \sigma_S, \rho),
\]
where:
\begin{itemize}
  \item $\mathbf{A} = \langle \mathcal{A}_1, \mathcal{A}_2, \dots, \mathcal{A}_K \rangle$
  is an ordered sequence of actions, with $\mathcal{A}_k \in \mathbb{A}$;
  \item $\sigma_S$ denotes the semantic description of the scenario;
  \item $\rho$ denotes the identifier of the associated requirement(s).
\end{itemize}
